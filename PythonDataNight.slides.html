Interactive data visualisations¶

Why should I care?

Get answers and tell stories¶

1 Make data look and 'feel' better

  • Amaze your clients and boss
  • Increase engagement

2 Make it easier to get answers

  • Explore important parts by moving, zooming, rotating
  • Highlight exact values at points of interest
  • Do all this without writing code!

3 Uncover hidden meaning

  • Change parameters and immediately see the changes
  • Chain visualisations together and see how they affect each other
  • Create streaming, animated or otherwise customised visuals

Interactivity is not the goal, but a means to an end.

Plot.ly¶

Plot.ly 'core':¶

  • Browser-based plot rendering system
  • Based on D3.js
  • Connectors for Python, R, Matlab, Julia, JavaScript
    • Converters from Matplotlib, ggplot2, etc.
  • Free, open-source
    • Yes, really.

Additionally:¶

  • Online hosting for plots & data
  • Visual plot editor (no coding)
  • Dashboard editor
  • Presentation editor
  • ...

Some of these require a premium account

def view_image(w):
    fig = FF.create_2D_density(

Closing thoughts¶

  • Plot.ly has many more features I couldn’t showcase
    • Curve fitting, on-premise hosted, other languages, etc.
  • You milage may vary
    • There are other tools that might be better suited for your needs
    • Spectrum goes from ‘versatile but difficult to program’ to ‘readily available but less customisable’
    • D3.js, bokeh, Vincent/Vega, ggplot, (gnuplot) ---> Tableau, Microsoft PowerBI, Grafana, Geckoboard, etc.
